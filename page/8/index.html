无分支获取取最小值

  • x < y, -(x < y) = -1, -1的二进制的所有位为1，(x ^ y) & -1 = (x ^ y), y ^ (x ^ y) = x

  • x > y, -(x < y) = 0, (x ^ y) & 0 = 0, y ^ 0 = y
