实现一个复数乘法

大部分时间都花在写正则匹配实部和虚部了

两个复数
$a + bi$, $c + di$

$(a + bi) * (c + di) = (ac - bd) + (bc + ad)i$

使用三个辅助变量

$$\begin{cases}
t_1 = (a - b) (c + d) \
t_2 = a d \
t_3 = b * c \
\end{cases},$$

$(a + bi) * (c + di) = (t_1 - t_2 + t_3) + (t_2 + t_3)i$

这样可以把四次乘法减少到三次
